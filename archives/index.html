Sublime Text3 TFS 安装与配置

  1. 首先在Sublime Text中安装好Sublime TFS插件
  2. 如果没有安装TFS Tools的话，选择安装TFS相应版本的TFS Power Tools

  3. 在Sublime TFS 插件设置中设置好tf_path与tftp_path,64 Win系统配置如下,可根据自己系统配置

  4. 基本上TFS常用功能都可以使用了,但是Get Latest Version 会报错说没有权限,此时的解决方法如下,打开Sublime TFS 插件的源码sublime_tfs.py文件,修改其中的get_latest与dir_get_latest方法,在get后加入/login:user,password登陆用户参数
